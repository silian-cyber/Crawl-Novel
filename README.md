# Crawl-Novel
一个用于抓取小说分类页数据，并进行简单数据分析和可视化的项目。

## 功能

1. **数据抓取**：使用Selenium爬取小说的基本信息（书名、作者、分类、简介、字数、状态、图片链接等），并保存到CSV文件。
2. **数据分析**：对抓取的数据进行清洗和可视化，包括分类分布、字数分布、状态比例、作者产出等分析，并生成词云。


## 环境依赖

见 `requirements.txt`，可通过以下命令安装：
```bash
pip install -r requirements.txt

注意事项
遵守网站规则爬虫脚本可能触发目标网站的反爬机制，使用时请务必遵守网站的 robots.txt 协议，控制请求频率（避免短时间内频繁访问），尊重网站的爬取限制和数据版权。
